Assignment name  : expand_str
Expected files   : expand_str.c
Allowed functions: write
--------------------------------------------------------------------------------

Write a program that takes a string and displays it with exactly three spaces
between each word, with no spaces or tabs at the beginning or the end of the
string, followed by a newline.

A word is a section of string delimited by spaces/tabs or by the start/end of
the string.

If the number of parameters is not 1, or if the string is empty, the program
displays a newline.

Examples:

$> ./expand_str " Let there    be light      " | cat -e
Let   there   be   light$
$> ./expand_str "It was the best of times" | cat -e
It   was   the   best   of   times$
$> ./expand_str "" | cat -e
$
$> ./expand_str | cat -e
$
