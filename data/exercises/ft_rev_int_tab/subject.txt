Assignment name  : ft_rev_int_tab
Expected files   : ft_rev_int_tab.c
Allowed functions: 
--------------------------------------------------------------------------------

Write a function that reverses a given array of integers in place
(the first element becomes the last, and so on). It takes a pointer to
an int and the number of elements as arguments.

Your function must be declared as follows:

void	ft_rev_int_tab(int *tab, int size);
