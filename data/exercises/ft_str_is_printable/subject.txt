Assignment name  : ft_str_is_printable
Expected files   : ft_str_is_printable.c
Allowed functions: 
--------------------------------------------------------------------------------

Create a function that returns 1 if the string contains only printable
characters, and 0 otherwise.
It must return 1 if str is empty.

Your function must be declared as follows:

int	ft_str_is_printable(char *str);
