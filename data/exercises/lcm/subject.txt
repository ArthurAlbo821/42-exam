Assignment name  : lcm
Expected files   : lcm.c
Allowed functions:
--------------------------------------------------------------------------------

Write a function who takes two unsigned int as parameters and returns the
computed LCM of those parameters.

LCM (Lowest Common Multiple) of two non zero integers is the smallest
positive integer divisible by both integers.

A LCM can be computed in two ways:

- You can compute every multiple of each integer until you have a common
  multiple other than 0.

- You can use the HCF (Highest Common Factor) of these integers and compute:
  LCM(x, y) = | x * y | / HCF(x, y)

If at least one parameter is 0, the LCM is 0.

Your function must be declared as follows:

unsigned int    lcm(unsigned int a, unsigned int b);
