Assignment name  : rpn_calc
Expected files   : rpn_calc.c
Allowed functions: atoi, printf, write, malloc, free
--------------------------------------------------------------------------------

Write a program that takes one string representing an arithmetic expression in
Reverse Polish Notation (RPN), evaluates it, and displays the result followed
by a newline.

The expression contains integer operands and the operators '+', '-', '*', '/'
and '%', all separated by spaces. In RPN, an operator comes after its two
operands.

If the number of arguments is not 1, or if the expression is invalid (not
enough operands, too many operands left, or an unknown token), the program
displays "Error" followed by a newline.

A division or modulo by zero will never be tested.

Examples:

$> ./rpn_calc "1 2 * 3 * 4 +" | cat -e
10$
$> ./rpn_calc "3 4 +" | cat -e
7$
$> ./rpn_calc "1 2 3 4 +" | cat -e
Error$
$> ./rpn_calc | cat -e
Error$
